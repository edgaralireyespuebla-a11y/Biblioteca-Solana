
Este proyecto consiste en el desarrollo de un **programa en la blockchain de Solana** que permite administrar una farmacia digital mediante operaciones básicas como **crear una farmacia, registrar medicamentos, consultar el inventario, modificar la disponibilidad y eliminar medicamentos**. El objetivo es demostrar cómo se pueden almacenar y gestionar datos dentro de la blockchain utilizando **cuentas del programa (Program Accounts)**.

Para desarrollarlo se utilizó **Rust** como lenguaje de programación junto con el framework **Anchor**, el cual facilita la creación de programas en Solana. El proyecto fue ejecutado y probado dentro de **Solana Playground**, un entorno en línea que permite escribir, compilar y desplegar programas sin necesidad de instalar herramientas localmente. También se utilizó **Devnet**, la red de pruebas de Solana, para realizar las transacciones utilizando tokens SOL de prueba.

Este proyecto forma parte de las actividades prácticas de la **certificación de desarrollo en Solana**, donde se aplican conceptos fundamentales como **Program Derived Addresses (PDA), cuentas en la blockchain, transacciones y pruebas de programas**, permitiendo comprender el funcionamiento básico de los **smart contracts en Solana**.
